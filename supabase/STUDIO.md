# Last-Price Supabase Studio

## Access Points

- **Studio Dashboard**: http://127.0.0.1:55323
- **API**: http://127.0.0.1:55321
- **Database**: postgresql://postgres:postgres@127.0.0.1:55322/postgres
- **Mailpit**: http://127.0.0.1:55324

## Project Info

- **Project ID**: `last-price`
- **Port Range**: 55321-55324 (different from default 54321-54324)

## Browser Tips

1. Bookmark the Studio URL as "Last-Price Supabase"
2. Use different browser profiles for each project
3. Check the URL port number to confirm which project you're in

## Quick Commands

```bash
# Start
pnpm dlx supabase start

# Status
pnpm dlx supabase status

# Stop
pnpm dlx supabase stop
```
